# Tibeuru Console — releases

Public installer downloads for **Tibeuru Console**.

[Download the latest installer](https://github.com/Tibeuru/tibeuru-console-releases/releases/latest).

## Install

Run the installer. Per-user install by default — no admin prompt. Elevate during install if you'd rather install for all users.
